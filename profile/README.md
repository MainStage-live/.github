# MainStage — Portable Live Performance Software

## About MainStage

MainStage provides a complete live performance environment for musicians playing on stage. MainStage combines virtual instruments, effects, backing tracks, and hardware control. MainStage enables users to create concert rigs with patches, splits, layers, and set lists. MainStage runs on desktop platforms. MainStage is trusted by keyboardists, guitarists, vocalists, electronic musicians, and touring professionals worldwide.

## Deep Dive — Live Performance Notes

Instruments: virtual synths, pianos, organs, samplers, drum machines
Effects: reverb, delay, compressor, EQ, distortion, modulation, amp sims
Concert layout: patches, sets, assignments, screen control, MIDI mapping
Backing tracks: play audio, loops, click track, count‑in, video
Hardware: MIDI controllers, audio interfaces, pedalboards, iPads (Logic Remote)

## Quick Reference for MainStage

| Operation | MainStage Action |
|-----------|------------------|
| Next patch | Right arrow |
| Previous patch | Left arrow |
| Next set | Cmd+Right |
| Previous set | Cmd+Left |
| Play backing track | Space |
| Stop backing track | Space (again) |
| Toggle metronome | Cmd+Shift+M |
| Show/hide layout | Cmd+L |
| Open edit mode | Cmd+E |
| Open concert | Cmd+O |
| Save concert | Cmd+S |
